# Changelog

## 1.5.0 — Recursive Evidence Substrate

### Added

- deterministic recursive identities and source-parameter fingerprints;
- orthographic controller positions;
- signed pairwise controller interval fields;
- bounded cognitive-length terms tied to recursive identity;
- trainer-integrated communication episodes;
- managed residual CSV artifact families;
- Staqtapp-TDS v3.5.2 result-first, reopen, CSV, scan-anchor, and Spiral integration;
- TDS-independent reasoning when persistence is disabled;
- release maturity boundary and artifact-integrity documentation.

### Corrected

- multi-output MSE gradient normalization;
- stability-controller routing from MC-4 to MC-3;
- route-message collisions;
- process-random spawned-controller IDs;
- mutable payloads inside frozen route messages;
- stale runtime release manifest;
- incomplete configuration validation;
- non-transactional coefficient updates;
- unbounded trainer, trace, route, and policy histories.

### Changed

- trace scoring now uses relational movement instead of a subjective novelty score;
- recursive depth is retained as evidence without a universal TDS depth penalty;
- TDS integration baseline moved from v2.6 to the v3.5.2 public API;
- POBSNN is explicitly identified as a working code name.

### Reserved

- pre-model discrete fold orientations;
- origamic unions and commutator axes;
- sine/cosine manifestation tuning;
- interval entropy and abstraction;
- prospective cognitive arrival;
- mature developmental personality.
